## I. Problem Definition
- Recommend restaurants based on user preferences and location. 

- Main requirements:
    - Accept location and food preferences (taste, price range,...) as input.
    - Return a list of top 3 restaurants matching the criteria.
## II.Decomposition of Sub-problem

| Component | Description | Expected Output |
|-----------|-------------|-----------------|
| Input user preferences | Preferred cuisines, price range, Dietary restrictions (vegan, halal,...) |  |
| Use user location | Typing by user or GPS coordinates| Normalized string of address considered as part of user input |
| Sort and filter restaurants | Based on user location or preferences | Ascendingly arranged list of restaurants for location or budget criteria |
| Recommend restaurants | After finding and sorting, select top 3 restaurants | Display on UI as a box text for each restaurant with name and address |
| Provide restaurant information | Additional details like price level, rating, time operation | Display on UI as a expandable section of each restaurant box |
| Format output in program | Structure the output for easy computing and processing | JSON format containing all relevant restaurant details |


## III. Proposed technology

| Component | Tool/Library | Note |
|-----------|-------------|-----------------|
| Input user preferences | Flask + HTML + CSS | Building typing bar for user input |
| Use user location | Flask + HTML + CSS / HTML5 Geolocation API | Building typing bar for address or track location automatically |
| Sort and filter restaurants | Sorting algorithm | Sorting base on returning information of API  |
| Recommend restaurants | Gemini API (generativeai)  | Consider input (location, preferences) as a prompt  |
| Provide restaurant information | Gemini API (generativeai) | The output API returns must have  |

## IV. Data Structures and Code Samples

### 1. Gemini API:
```python
import google.generativeai as genai
from dotenv import load_dotenv
import os

load_dotenv()
API_KEY = os.getenv("GOOGLE_API_KEY")

genai.configure(api_key=API_KEY)

model = genai.GenerativeModel('gemini-2.5-flash')
prompt = input("Enter a prompt: ")
response = model.generate_content(
    prompt, 
    generation_config = {
        "response_mime_type": "application/json",
        "response_schema": schema
    }                              
)
```

### 2. Format ouput in JSON:
```python
import google.ai.generativelanguage as glm
schema = glm.Schema(
    type = glm.Type.ARRAY,
    items = glm.Schema(
        type = glm.Type.OBJECT,
        properties = {
            "Restaurant": glm.Schema(type = glm.Type.STRING),
            "Address": glm.Schema(type = glm.Type.STRING),
        },
        required = ["Restaurant", "Address"]
    )
)
```
